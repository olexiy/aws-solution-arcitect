## Installing AWS CLI 
```bash
curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
unzip awscliv2.zip
sudo ./aws/install
```
for this project AWS CLI install is hadled with Gitpod config yaml
